# Proyecto-de-ingenier-a-de-software
NATADINATA – Sistema de Facturación y Gestión de Ventas
Descripción

NATADINATA es un sistema de software diseñado para la gestión de ventas, facturación, inventario y clientes dentro de una empresa.
El sistema permite a los usuarios iniciar sesión, administrar información de la empresa, registrar ventas del día, generar facturas y controlar productos mediante un sistema organizado de módulos.

Este proyecto fue desarrollado como parte de la asignatura de Ingeniería de Software, aplicando conceptos de análisis, diseño de interfaces y estructuración de sistemas.

Objetivo del proyecto
Desarrollar un sistema que permita gestionar de manera eficiente las ventas y la facturación de una empresa, facilitando el control de productos, clientes y transacciones.
Funcionalidades principales
Autenticación de usuarios

Inicio de sesión mediante:
Usuario
Contraseña
Opción de crear nuevos usuarios o empleados

Registro de empresa:
Permite registrar la información de la empresa:
Nombre de la empresa
Dirección
Ciudad
NIT
Esta información se utiliza para la generación de facturas.

Panel principal
Después del inicio de sesión, el sistema muestra un panel de control con las siguientes secciones:
Ventas
Contabilidad
Inventario
Clientes
Configuración
Ajustes

También incluye:
Buscador
Historial de transacciones
Ventas del día
Totales y ventas realizadas

Generación de facturas
El sistema permite crear facturas con la siguiente información:
Nombre del producto
Cantidad
Precio
Fecha de facturación
Total de la venta

Opciones de pago:
Pago total
Abono
No pago
Las facturas generadas pueden confirmarse o cancelarse.

Gestión de productos
En la ventana de facturación se pueden agregar productos con:
Nombre
Cantidad
Precio
Esto permite registrar las ventas realizadas.

Flujo del sistema

1️ Usuario inicia sesión
2️ Accede al panel principal
3️ Registra o consulta ventas del día
4️ Genera una factura
5️ Agrega productos a la factura
6️ Selecciona el tipo de pago
7️ Confirma o cancela la factura

Diseño de interfaz
El sistema incluye interfaces para:
Inicio de sesión
Panel de control
Registro de empresa
Ventana de ventas
Generación de facturas

El diseño está enfocado en facilidad de uso y organización modular.
